Defines properties that control input masking and formatting behavior for a text-based editor.

The ITextEditorMaskProperties interface is implemented by the UI library and is not intended to be implemented by consumer code. 

public interface ITextEditorMaskProperties

Examples

The following example demonstrates configuring a text editor mask:

ITextInputEditor editor = factory.CreateTextEdit();

editor.MaskProperties.MaskType = EditMaskType.Simple;
editor.MaskProperties.MaskExpression = @"(\+91)-0000-0000-00"; // phone number format

Remarks

This interface provides a way to restrict and format user input using mask expressions and predefined mask types.

Masks can be used to enforce input patterns (for example, phone numbers, dates, or custom formats) and to control how text is interpreted and displayed.

The combination of MaskType and MaskExpression determines how the input is validated and formatted.

The lifetime of an ITextEditorMaskProperties instance is managed by the associated editor. Consumers typically do not create or dispose this object directly.

Properties

MaskExpression

Gets or sets the mask expression used to control input formatting and validation.

string MaskExpression { get; set; }

Property Value

string

A string that defines the mask pattern.

Remarks

The interpretation of this value depends on MaskType.

Examples:

  • (000) 000-0000 for a phone number
  • dd-MM-yyyy for a date-like pattern

If no mask is required, this property can be null or an empty string.

MaskType

Gets the type of mask applied to the editor.

EditMaskType MaskType { get; }

Property Value

EditMaskType

An EditMaskType value that determines how the mask expression is interpreted and applied.

Remarks

The mask type defines the rules used to validate and format user input.

Together with MaskExpression, this property controls how input is processed, such as restricting characters, enforcing patterns, or applying date/time and numeric formats.

The value is typically defined by the editor implementation and may not be changeable by consumers.

ValueType

Gets the data type of the value produced by the editor after applying the mask.

Type ValueType { get; }

Property Value

Type

A Type that specifies the resulting value type (for example, string, int, or DateTime).

Remarks

This property indicates how the editor's input should be interpreted after mask processing.

The value is typically determined by the editor implementation and mask type, and is not intended to be modified by consumers.
