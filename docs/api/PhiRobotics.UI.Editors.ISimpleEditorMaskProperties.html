Defines properties that control simple masking behavior for text-based editors.

The ISimpleEditorMaskProperties interface is implemented by the UI library and is not intended to be implemented by consumer code.

Examples

The following example demonstrates configuring a simple editor mask:

// Phone number mask: (+91)-0000-0000-00
ITextInputEditor editor = inputEditorFactory.CreateTextEdit();

editor.Configure(null, EditMaskType.Simple);
ISimpleEditorMaskProperties maskProperties = 
    (ISimpleEditorMaskProperties)editor.MaskProperties;

maskProperties.Placeholder = '_';
maskProperties.AllowBlankInput = false;
maskProperties.SaveLiterals = true;

maskProperties.MaskExpression = @"(\+91)-0000-0000-00";

Remarks

This interface extends ITextEditorMaskProperties to provide position-based masking functionality.

Simple masks allow users to enter alphabetic, numeric, or alphanumeric characters at predefined positions using mask symbols.

These properties control how placeholder characters are displayed, whether blank input is permitted, and how literal characters are handled in the editor value.

The lifetime of an ISimpleEditorMaskProperties instance is managed by the associated editor. Consumers typically do not create or dispose this object directly.

Properties

AllowBlankInput

Gets or sets whether blank input is allowed.

bool AllowBlankInput { get; set; }

Property Value

bool

true if blank input is allowed; otherwise, false.

Remarks

When disabled, the editor requires input that satisfies the defined mask expression.

Placeholder

Gets or sets the character displayed for empty mask positions.

char Placeholder { get; set; }

Property Value

char

A char that specifies the placeholder character.

Remarks

This character is shown in positions where input is required but not yet provided.

SaveLiterals

Gets or sets whether literal (non-editable) characters are included in the editor value.

bool SaveLiterals { get; set; }

Property Value

bool

true to include literals in the value; otherwise, false.

Remarks

Literal characters are fixed parts of the mask that cannot be edited by the user.

For example, if the mask is ID: 00000:

  • If disabled, only the digits (for example, 12345) are stored as the EditValue.
  • If enabled, the full formatted value (for example, ID: 12345) is stored.
