Represents a collection that stores images and allows them to be retrieved using unique keys.

The IImageCollection interface is implemented by the UI library and is not intended to be implemented by consumer code. 

public interface IImageCollection

Remarks

This collection is used by ITree to resolve images associated with nodes. Images are registered in the collection using a unique key and can later be referenced by that key (for example, through ImageKey or ExpandedImageKey).

Using a centralized image collection allows multiple nodes to reuse the same image without duplicating resources.
