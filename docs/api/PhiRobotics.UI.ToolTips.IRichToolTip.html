Represents a rich tooltip that can display multiple content regions, such as text, images, titles, separators, and footers.

This interface is implemented by the PhiRobotics UI Library and is not intended to be implemented by consumer code.

Examples

The following example demonstrates how to create and configure a rich tooltip. The tooltip includes a title, descriptive content with an image, a separator, and a footer section.

private IRichToolTip CreateToolTipForFileOpenButton(IToolTipFactory toolTipFactory)
{
    IRichToolTip toolTip = toolTipFactory.CreateToolTip();   

    // Add the main title
    IRichToolTipTitleItem title = toolTip.Items.AddTitle("Open File Dialog");

    // Add descriptive content with an image
    IRichToolTipItem content = toolTip.Items.Add("Show the Open file dialog box.");
    content.Image = Resources.SaveFileDialog_Thumbnail;

    // Add a separator to group sections
    IRichToolTipSeparatorItem separator = toolTip.Items.AddSeparator();

    // Add a footer with additional information and an image
    IRichToolTipTitleItem footer = toolTip.Items.AddTitle("For more information see help.");
    footer.Image = Resources.Help;

     return toolTip;
}

Remarks

A rich tooltip is composed of multiple regions (items); title, content, separator, and footer, allowing you to build structured and visually enhanced tooltip layouts.

Each region in the tooltip is represented by an item in the Items collection.

Consumer code can obtain an instance of IRichToolTip using the CreateToolTip() method. The lifetime of the tooltip instance is managed by the consumer.

AllowHtmlText

Gets or sets whether HTML formatting is enabled for this tooltip.

bool AllowHtmlText { get; set; }

Property Value

bool

true to allow HTML formatting; otherwise, false.

Remarks

This setting can override the tooltip controller's HTML formatting setting for this specific tooltip.

DistanceBetweenItems

Gets or sets the spacing between tooltip items (such as title, content, and footer).

int DistanceBetweenItems { get; set; }

Property Value

int

An int value, in pixels, that specifies the distance between items.

FixedTooltipWidth

Gets or sets whether the tooltip uses a fixed width.

bool FixedTooltipWidth { get; set; }

Property Value

bool

true if the tooltip width is fixed; otherwise, false to automatically size the tooltip based on its content.

Remarks

When set to false, the tooltip width is calculated automatically. Use the MaxWidth property to limit the maximum width.

Items

Gets the collection of items that define the content regions of the tooltip.

IToolTipItemCollection Items { get; }

Property Value

IToolTipItemCollection

An IToolTipItemCollection containing items such as titles, text blocks, separators, and footers.

Remarks

The order of items in the collection determines how they are rendered within the tooltip.

MaxWidth

Gets or sets the maximum width of the tooltip, in pixels.

int MaxWidth { get; set; }

Property Value

int

An int value that specifies the maximum width.

Remarks

This property is primarily used when FixedTooltipWidth is set to false, allowing the tooltip to resize dynamically while respecting this limit.

OwnerAllowHtmlText

Gets whether HTML formatting is allowed by the tooltip controller that owns this tooltip.

bool OwnerAllowHtmlText { get; }

Property Value

bool

true if HTML formatting is allowed by the owner; otherwise, false.

Remarks

HTML formatting behavior follows this priority:

  • AllowHtmlText – global setting for all tooltips.
  • AllowHtmlText – overrides the controller setting for this tooltip.
  • AllowHtmlText – overrides the tooltip setting for a specific item.

Padding

Gets or sets the padding between the tooltip content and its outer boundary.

Padding Padding { get; set; }

Property Value

Padding

A Padding value that defines the inner spacing.
