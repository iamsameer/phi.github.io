Defines a factory for creating alert panels and related components.

The IAlertPanelFactory interface is implemented by the UI library and is not intended for consumer implementation. 

public interface IAlertPanelFactory

Remarks

Consumer code uses this interface to create instances of IAlertPanel and IButton for displaying notifications and alerts.

Consumer code obtains an instance via the AlertPanelFactory property. The lifetime of the factory instance is managed by the UI library.
