Remarks

The WizardIdAttribute is used to mark a class as a wizard and associate it with a unique wizard identifier.

During wizard discovery, the wizard host scans assemblies marked with ExportedWizardAttribute and inspects their types for the presence of this attribute. Only classes decorated with WizardIdAttribute are considered valid wizard implementations.

The specified identifier uniquely identifies the wizard. If a wizard with the same identifier has already been registered, the duplicate wizard will be ignored during registration.

The following example shows how to declare a wizard class and associate it with a wizard identifier. 
[WizardId("CreateProjectWizard")]
public class CreateProjectWizard : IWizard
{
    public WelcomePage WelcomePage => 
        new WelcomePage("Create Project", "This wizard will guide you through creating a new project.");

    public CompletionPage CompletionPage => 
        new CompletionPage("Completed", "The project has been created successfully.");

    public IReadOnlyDictionary<string, IWizardPage> Pages { get; }

    public void Initialize(object data) { }
    public void Cancel() { }
    public object Complete() => null;
    public void Activated(IWizardPage page) { }
}
