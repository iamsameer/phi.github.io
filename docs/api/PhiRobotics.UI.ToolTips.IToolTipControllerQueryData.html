Provides data for the QueryObjectInfo event. event.

public interface IToolTipControllerQueryData

Remarks

This data object allows event handlers to inspect and modify the tooltip information before it is displayed.

The instance of IToolTipControlInfo is provided via the Info property and can be modified or replaced by event handlers.
