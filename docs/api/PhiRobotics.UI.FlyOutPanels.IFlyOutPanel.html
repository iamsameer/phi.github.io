Namespace

Represents a fly-out panel that can be attached to a control and displayed dynamically.

The IFlyOutPanel interface is implemented by the PhiRobotics UI library and is not intended to be implemented by consumer code. 

public interface IFlyOutPanel : IDisposable, ISynchronizeInvoke, IWin32Window, IComponent, IDisposable

Examples

The following example demonstrates how to create and display a fly-out panel:

// Create a new fly-out panel (without initial content)
IFlyOutPanel flyOutPanel = flyOutPanelFactory.CreatePanel(null);

// Create and assign content
Control content = new CustomControl();
content.Size = new Size(250, 100);
flyOutPanel.Content = content;

// Attach the panel to a host control
flyOutPanel.AttachTo(flyOutHost);

// Display the panel
flyOutPanel.ShowPanel();

Remarks

Instances of IFlyOutPanel are created using the CreatePanel(Control) or CreatePanel(Control, FlyOutPanelOptions) method.

The lifetime of an IFlyOutPanel instance is managed by the consumer. The consumer is responsible for disposing the panel when it is no longer needed.
