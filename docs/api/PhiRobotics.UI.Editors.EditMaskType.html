Specifies the type of input mask applied to an editor.

public enum EditMaskType

Fields

DateTime = 1

Specifies a mask for entering date and time values.

The mask expression defines the expected date-time format (for example, yyyy-MM-dd).

None = 0

No mask is applied.

The editor accepts unrestricted input.

Numeric = 2

Specifies a mask that restricts input to numeric values.

The mask expression may define numeric formatting such as decimal precision.

RegEx = 3

Specifies a mask based on an extended regular expression pattern.

This type supports advanced pattern matching using extended regular expressions.

Regular = 4

Specifies a mask based on a standard regular expression pattern.

This type uses regular expressions to validate input.

Simple = 5

Specifies a simple positional mask.

The mask expression defines required character types at specific positions (for example, digits, letters, or alphanumeric characters).

TimeSpan = 6

Specifies a mask for entering time interval values.

The mask expression defines the format of the time span (for example, hh:mm:ss).

Remarks

The EditMaskType enumeration defines how an input mask expression is interpreted and applied by an editor.

It is used with MaskType to control input validation and formatting behavior.

Each mask type provides different rules for how user input is constrained, validated, and displayed.
