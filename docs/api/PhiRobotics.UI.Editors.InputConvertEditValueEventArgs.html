Handled

Gets or sets a value specifying whether default edit value conversion/formatting is required.

Declaration
public bool Handled { get; set; }
Property Value
Type Description
bool

true to prohibit default edit value conversion/formatting; otherwise, false.
Remarks

Events used to convert and format the edit value ParseEditValue fire when the end-user accepts input. The Handled property specifies whether default conversion/formatting should be performed. By default, this property value shall be false. Hence, you must set the Handled property value to true when it is necessary to override default edit value conversion/formatting.

note

if the Handled property value is false and the Value property has been modified, the default formatting/parsing algorithm will be applied to the value specified by the Value property.
