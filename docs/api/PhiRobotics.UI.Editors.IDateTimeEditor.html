Remarks

The IDateTimeEditor interface extends IInputEditor and provides functionality for editing date and time values.

This editor supports structured input using date-time masks, allowing users to enter and modify individual date and time components such as year, month, day, hours, minutes, and seconds.

Use the MaskProperties property to configure the input mask and control how date-time values are entered and displayed.

Instances of IDateTimeEditor are created using the IInputEditorFactory.

Examples

The following example demonstrates how to configure a date-time editor:

IDateTimeEditor editor = factory.CreateDateTimeEdit();

editor.NullValuePrompt = "Enter date...";
editor.NullValuePromptForeColor = Color.LightSkyBlue;

IDateTimeEditorMaskProperties maskProperties = editor.MaskProperties;
maskProperties.UseAdvancingCaret = true;
maskProperties.SpinWithCarry = true;
maskProperties.MaskExpression = "f";
