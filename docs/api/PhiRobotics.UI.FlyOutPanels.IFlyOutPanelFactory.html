Provides factory methods for creating IFlyOutPanel instances.

The IFlyOutPanelFactory interface is implemented by the PhiRobotics UI library and is not intended for consumer implementation. 

public interface IFlyOutPanelFactory

Remarks

Use this interface to create instances of IFlyOutPanel.

An instance of IFlyOutPanelFactory can be obtained through the FlyOutPanelFactory property.

The lifetime of the factory instance is managed by the library.
