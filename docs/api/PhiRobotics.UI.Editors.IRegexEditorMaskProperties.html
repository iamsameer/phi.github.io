Defines properties that control regular expression-based masking behavior for text-based editors.

The IRegexEditorMaskProperties interface is implemented by the UI library and is not intended to be implemented by consumer code.

Examples

The following example demonstrates configuring a regex editor mask:

// Email address mask
ITextInputEditor editor = inputEditorFactory.CreateTextEdit();

editor.Configure(null, EditMaskType.RegEx);
IRegexEditorMaskProperties maskProperties = 
    (IRegexEditorMaskProperties)editor.MaskProperties;

maskProperties.Placeholder = '_';
maskProperties.AllowBlankInput = false;

string expression = @"((([0-9a-zA-Z_%-])+[.])+|([0-9a-zA-Z_%-])+)+@phi-robotics\.com";
maskProperties.MaskExpression = expression;

Remarks

This interface extends ITextEditorMaskProperties to provide advanced pattern-based validation using regular expressions.

It allows you to define flexible and complex input rules where the entered text must match a specified regular expression pattern.

These properties control placeholder behavior, blank input handling, and optional auto-complete features that assist users during input.

The lifetime of an IRegexEditorMaskProperties instance is managed by the associated editor. Consumers typically do not create or dispose this object directly.

AllowBlankInput

Gets or sets whether blank input is allowed.

bool AllowBlankInput { get; set; }

Property Value

bool

true if blank input is allowed; otherwise, false.

Remarks

When disabled, the input must satisfy the defined regular expression.

IsAutoComplete

Gets or sets whether auto-complete suggestions are enabled.

bool IsAutoComplete { get; set; }

Property Value

bool

true to enable auto-complete functionality; otherwise, false.

Remarks

When enabled, the editor can suggest values that match the regular expression as the user types.

IsOptimistic

Gets or sets whether auto-complete uses optimistic matching behavior.

bool IsOptimistic { get; set; }

Property Value

bool

true to suggest the shortest matching value; otherwise, false.

Remarks

This property is effective only when IsAutoComplete is true.

When enabled, the editor attempts to complete input using the shortest valid match.

Property Value

char

A char that specifies the placeholder character.

Remarks

This character is shown in positions where input is expected but not yet provided.

This property controls the visual representation of incomplete input.
