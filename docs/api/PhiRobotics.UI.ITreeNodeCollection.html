Interface ITreeNodeCollection

Represents a collection of ITreeNode instances within an ITree.

The ITreeNodeCollection interface is implemented by the UI library and is not intended to be implemented by consumer code.

public interface ITreeNodeCollection : IEnumerable<ITreeNode>, ICollection, IEnumerable
Remarks

A node collection manages hierarchical relationships in a tree:

  • Root-level nodes are accessed through the RootNodes property.
  • Each ITreeNode exposes its child nodes through the Nodes property.

The collection provides methods to append, remove, and enumerate nodes.
