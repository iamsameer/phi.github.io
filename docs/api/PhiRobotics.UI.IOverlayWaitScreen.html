Interface IOverlayWaitScreen

Describes an interface which is used to provide a semi-transparent splash screen that can overlaps provided a control or form.

Inherited Members

Syntax
public interface IOverlayWaitScreen : IDisposable, IDisposable
Remarks

Following are the features of IOverlayWaitScreen : overlaps a control or form; prevents users from interacting with the overlapped control; overlaps the control even if it changes its size or position on the screen; runs in a separate thread and does not lock the main and operation threads;

QueueCloseUpAction(Action)

Specifies an action executed when the overlay screen is closed.

Declaration
void QueueCloseUpAction(Action closeUpAction)
Parameters
Type Name Description
Action closeUpAction

An action executed when the Overlay Screen is closed.
Remarks

If this method is called more than once, only the last action added to queue will be executed when the overlay screen is closed, and all other actions will be ignored. If this method is called with a null argument, any previously set action will be cleared from the queue.
