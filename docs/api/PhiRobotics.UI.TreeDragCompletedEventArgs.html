public class TreeDragCompletedEventArgs : EventArgs
Remarks

This event is raised after a drag-and-drop operation has finished.

It allows the subscriber to perform post-operation logic, such as:

  • Removing original nodes after a move operation.
  • Updating UI state.
  • Cleaning up temporary data.
