Interface ITreeNode

Represents a node within a ITree control.

The ITreeNode interface is implemented by the UI library and is not intended to be implemented by consumer code.

Namespace: PhiRobotics.UI
Assembly: PhiRobotics.UI.dll
Syntax
public interface ITreeNode
Remarks

The ITreeNode interface defines properties for interacting with a node in a hierarchical tree structure.

Nodes are organized using parent-child relationships. Each node can contain child nodes exposed through the Nodes collection.

Root nodes are stored in the RootNodes collection and do not have a parent; therefore, their ParentNode property returns null.

End users can expand a node using its expand button. The HasChildren property determines whether this button is displayed.

You can set HasChildren to true even when no child nodes exist to support dynamic loading scenarios. When the user expands such a node, the OnBeforeExpandNode(ITree, BeforeExpandNodeEventArgs) event is raised, allowing child nodes to be loaded on demand.
