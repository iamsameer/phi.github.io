Remarks

This event is raised when data is dropped within the bounds of the ITree control.

The subscriber is responsible for:

  • Validating the dropped Data.
  • Determining the target node using TreeHitTestData.
  • Performing the appropriate operation (Move, Copy, etc.).
  • Setting DropEffect to indicate the final result.

If DropEffect is set to None, the operation is treated as canceled.
