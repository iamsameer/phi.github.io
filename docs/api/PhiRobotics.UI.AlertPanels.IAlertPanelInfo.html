Contains information to be displayed in an alert window.

public interface IAlertPanelInfo

Remarks

To show an alert window displaying a specific information, you can create an IAlertPanelInfo object with information to be displayed, and pass it to the Show(Form, IAlertPanelInfo, bool) method. An IAlertPanelInfo object’s properties allow you to specify the caption, text, and image to be displayed in alert panel. In addition, you can set custom data to the Tag property.

You can subsequently use this data while processing events for alert panels (BeforeFormShown or PanelTextClick).

Objects implementing IAlertPanelInfo are passed to Show(Form, IAlertPanelInfo, bool) to display notifications. They allow customization of the alert's text, caption, images, and custom metadata.
