Remarks

This event is raised when a drag operation is initiated from the ITree control.

The subscriber is responsible for:

  • Creating and assigning an IDataObject to DataObject.
  • Specifying allowed drag operations using AllowedEffects.

If AllowedEffects is set to None, the drag operation is canceled.
