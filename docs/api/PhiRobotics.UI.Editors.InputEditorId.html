Examples

The following example demonstrates how to create an editor using an identifier:

IInputEditorFactory factory = GetFactory();

IInputEditor editor = factory.Create(InputEditorId.TextEdit);

Remarks

InputEditorId is a lightweight value object used to uniquely identify a specific type of input editor (for example, text, boolean, numeric, or date editors).

It is primarily used with IInputEditorFactory to request the creation of a specific editor type via Create(InputEditorId).

Predefined identifiers are provided for common editor types.
