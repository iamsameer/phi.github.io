Describes properties of a time span editor mask that allows users to enter time intervals.

The ITimeSpanEditorMaskProperties interface is implemented by the UI library and is not intended to be implemented by consumer code.

Examples

The following examples demonstrate configuring a time span editor mask:

// Using a text editor with a time span mask
ITextInputEditor editor = inputEditorFactory.CreateTextEdit();

editor.Configure(null, EditMaskType.TimeSpan);
ITimeSpanEditorMaskProperties maskProperties = 
    (ITimeSpanEditorMaskProperties)editor.MaskProperties;

maskProperties.UseAdvancingCaret = true;
maskProperties.SpinWithCarry = true;
maskProperties.MaskExpression = "mm MM, ss SS";

// Using a dedicated date-time editor
IDateTimeEditor dateTimeEditor = inputEditorFactory.CreateTimeSpanEditor();
ITimeSpanEditorMaskProperties maskProperties = dateTimeEditor.MaskProperties;

maskProperties.UseAdvancingCaret = true;
maskProperties.SpinWithCarry = true;
maskProperties.MaskExpression = "mm MM, ss SS";

Remarks

The ITimeSpanEditorMaskProperties interface extends ITextEditorMaskProperties and provides additional settings specific to time interval input.

A time span mask enables structured input for duration values such as days, hours, minutes, and seconds. It supports segmented input, allowing users to edit individual parts of a time interval independently.

Use this mask to configure behaviors such as segment navigation, caret movement, value propagation between segments, and handling of negative time values.

This mask type is supported by both ITextInputEditor and ITimeSpanEditor in the UI library.

The lifetime of an ITimeSpanEditorMaskProperties instance is managed by the associated editor. Consumers typically do not create or dispose this object directly.

Properties

AllowNegativeValues

Gets or sets whether users can enter negative time offsets.

bool AllowNegativeValues { get; set; }

Property Value

bool

true if negative values are allowed; otherwise, false.

Remarks

If disabled and a negative value is assigned directly to the EditValue property, the value remains negative until the user modifies it, after which it is converted to a positive value.

DefaultPart

Gets or sets the default segment selected when the editor receives focus.

TimeSpanMaskPart? DefaultPart { get; set; }

Property Value

TimeSpanMaskPart?

A nullable TimeSpanMaskPart value that specifies the initially selected segment.

EnableMnemonics

Gets or sets whether users can use keyboard mnemonics to navigate between segments.

bool EnableMnemonics { get; set; }

Property Value

bool

true to enable mnemonics; otherwise, false.

Remarks

For example, in the "en-US" culture:

  • Press d for days
  • Press h for hours
  • Press m for minutes
  • Press s for seconds
Users can type h5 to set the hours segment regardless of the caret position.

HideInsignificantPartsOnly

Specifies whether the editor should hide middle optional segments with zero values.

bool HideInsignificantPartsOnly { get; set; }

Property Value

bool

true to hide the insignificant parts with zero values; otherwise false.

InputMode

Gets or sets the input mode for the time span editor.

TimeSpanMaskInputMode InputMode { get; set; }

Property Value

TimeSpanMaskInputMode

A TimeSpanMaskInputMode value that specifies how input is handled.

ResetPartsOnSelectAllInput

Gets or sets whether all segments (except the default segment) reset when the user replaces the entire value.

bool ResetPartsOnSelectAllInput { get; set; }

Property Value

bool

true to reset segments on full input; otherwise, false.

SpinWithCarry

Gets or sets whether adjacent segments are automatically adjusted when spinning values.

bool SpinWithCarry { get; set; }

Property Value

bool

true to propagate changes to neighboring segments; otherwise, false.

Remarks

For example, increasing seconds past their maximum value increments the minutes segment.

UseAdvancingCaret

Gets or sets whether the caret automatically moves to the next segment when the current one is filled.

bool UseAdvancingCaret { get; set; }

Property Value

bool

true to advance the caret automatically; otherwise, false.
