Defines a factory for creating instances of ISvgImage.

The ISVGFactory interface is implemented by the UI library and is not intended for consumer implementation. 

public interface ISvgFactory

Remarks

The ISVGFactory interface provides methods to instantiate SVG images from various sources, such as file paths or streams.

Consumer code obtains an instance via the SVGFactory property. The lifetime of the factory instance is managed by the UI library.
