Interface IRichToolTipItem

Represents a tooltip item that displays text and optional visual content (such as images or icons) within a rich tooltip.

This interface is implemented by the PhiRobotics UI library and is not intended to be implemented by consumer code.

Remarks

This item is used to divide content into logical sections for better readability.

This item is typically used to display descriptive content in the tooltip body.

A rich tooltip item can include text, images, icons, or SVG content, and provides layout customization options such as alignment, spacing, and indentation.
