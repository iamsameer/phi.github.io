Namespace PhiRobotics.UI.Dialogs

Interfaces

ICommonDialogFactory

Provides factory methods for creating common dialog components such as message boxes, input boxes, and login dialogs.

The ICommonDialogFactory interface is implemented by the UI library and is not intended to be implemented by consumer code.

IInputBox

Represents a modal dialog that allows the user to enter a value using a single editor.

The IInputBox interface is implemented by the UI library and is not intended to be implemented by consumer code.

IInputBoxArgs

Represents configuration settings for an IInputBox.

ILoginDialog

Represents a user interface contract for a login dialog, providing support for credential input and dialog behavior.

The ILoginDialog interface is implemented by the UI library and is not intended to be implemented by consumer code.

IMessageBox

Represents a dialog used to display informational, warning, or confirmation messages.

The IMessageBox interface is implemented by the UI library and is not intended to be implemented by consumer code.

IMessageBoxArgs

Represents configuration settings for an IMessageBox.
