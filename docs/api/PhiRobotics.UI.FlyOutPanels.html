FlyOutPanelOptions

Represents a set of configuration options used to initialize an IFlyOutPanel.

Interfaces

IFlyOutPanel

Represents a fly-out panel that can be attached to a control and displayed dynamically.

The IFlyOutPanel interface is implemented by the PhiRobotics UI library and is not intended to be implemented by consumer code.

IFlyOutPanelFactory

Provides factory methods for creating IFlyOutPanel instances.

The IFlyOutPanelFactory interface is implemented by the PhiRobotics UI library and is not intended for consumer implementation.
