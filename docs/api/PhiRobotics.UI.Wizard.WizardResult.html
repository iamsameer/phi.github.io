Remarks

The WizardResult class encapsulates the outcome of a wizard dialog execution. It contains both the dialog result indicating how the wizard was closed and any data produced by the wizard during execution.

The Result property indicates whether the wizard completed successfully, was cancelled by the user, or was aborted due to an error.

When Result is OK, the Data property typically contains the result data generated by the wizard. For other dialog results, this value is usually null.
