Interface IColourEditor

Represents an editor used to view and modify Color values.

The IColourEditor interface is implemented by the UI library and is not intended to be implemented by consumer code.

Remarks

This editor provides a user interface for selecting color values. It typically allows users to choose colors using common mechanisms such as a color picker.

The editor can support different color selection modes, including solid colors, system colors, or custom-defined values, depending on the implementation.

The lifetime of an IColourEditor instance is managed by the consumer. The consumer is responsible for disposing the editor when it is no longer needed.

Examples

The following example demonstrates how to create a color editor:

IColourEditor editor = factory.CreateColourEdit();
