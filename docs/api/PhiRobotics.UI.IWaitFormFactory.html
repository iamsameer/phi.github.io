Parameters

properties WaitFormProperties

Properties used to create IWaitForm.

owner Form

Returns

Remarks

Implementation of this method a copy of the properties object before using or passing it to the wait form. This ensures that the original 'WaitFormProperties' object is not modified and can be reused.

ArgumentNullException

Thrown, if specified properties or owner is null.

InvalidOperationException

Thrown, if specified owner's IsDisposed property returns true.
