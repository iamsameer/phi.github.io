Remarks

This event is raised when dragged data enters or moves within the bounds of the ITree control.

The subscriber should:

  • Validate the dragged Data.
  • Inspect the target location using TreeHitTestData.
  • Evaluate KeyState to determine user intent.
  • Assign an appropriate value to Effect.

Setting Effect to None prevents dropping.
