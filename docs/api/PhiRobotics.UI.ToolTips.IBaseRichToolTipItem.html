Defines the base contract for all rich tooltip items.

This interface provides common properties and functionality shared by all tooltip item types, such as text items, separators, and title/footer items. 

public interface IBaseRichToolTipItem

Remarks

Tooltip items are building blocks of a rich tooltip. Each item represents a distinct visual region within the tooltip layout.
