Provides access to UI-related factory instances.

The IUIFactoryService interface is implemented by the UI library and is not intended for consumer implementation. 

public interface IUIFactoryService

Remarks

This service acts as a centralized entry point for obtaining factory instances used to create UI components and resources.

Use this service to access factories for creating alert panels, dialogs, editors, tooltips, tree controls, wizards, and other UI elements.
