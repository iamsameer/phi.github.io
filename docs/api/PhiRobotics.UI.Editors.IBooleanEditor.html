Describes an interface which defines an editor that is used to view and edit boolean values.

The IBooleanEditor interface is implemented by the UI library and is not intended to be implemented by consumer code.

Examples

The following example demonstrates how to configure a boolean editor:

IBooleanEditor editor = factory.CreateBooleanEdit();

editor.Caption = "Enable Feature";
editor.AllowIntermediate = true;
editor.Style = CheckBoxStyle.CheckBox;

editor.TrueValue = "Yes";
editor.FalseValue = "No";
editor.IntermediateValue = "Unknown";

Remarks

IBooleanEditor provides functionality for editing values that represent logical states such as true, false, and optionally an intermediate (indeterminate) state.

The visual representation of the editor can vary depending on the selected Style. Common styles include a check box, radio button, or other customized appearances.

In addition to standard boolean behavior, this editor allows mapping between logical states and values via TrueValue, FalseValue, and IntermediateValue.

Use the AllowIntermediate property to enable or disable the indeterminate state.

The lifetime of an IBooleanEditor instance is managed by the consumer. The consumer is responsible for disposing the editor when it is no longer needed.

Properties

AllowIntermediate

Gets or sets a value indicating whether the editor supports three states: Unchecked, Checked, and Indeterminate.

bool AllowIntermediate { get; set; }

Property Value

bool

true if the editor supports an intermediate state; otherwise, false.

Remarks

When set to false, the editor behaves as a standard two-state boolean editor.

Gets or sets the caption text associated with the editor.

string Caption { get; set; }

A string that represents the text displayed alongside the editor.

object FalseValue { get; set; }

Property Value

object

An object representing the value associated with false.

FalseValueImage

Property Value

IntermediateValue

Gets or sets the value that corresponds to the intermediate (indeterminate) state. (i.e nor true neither false,).

object IntermediateValue { get; set; }

Property Value

object

An object representing the value associated with the intermediate state.

Remarks

This property is only used when AllowIntermediate is true.

IntermediateValueImage

Gets or sets the image displayed instead of the default check box when the editor represents the intermediate (indeterminate) value.

Image IntermediateValueImage { get; set; }

Property Value

Image

An Image that replaces the default visual representation for the intermediate state.

Remarks

This property is only relevant when AllowIntermediate is true.

Style

Gets or sets the visual style used to represent the boolean editor.

CheckBoxStyle Style { get; set; }

Property Value

CheckBoxStyle

A CheckBoxStyle value that determines how the editor is rendered.

Remarks

The style affects how users interact with the editor, such as whether it appears as a check box, radio button, or a custom style.

object

An object representing the value associated with true.
