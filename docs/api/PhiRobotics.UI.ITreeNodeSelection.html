Represents the current selection within an ITree.

The ITreeNodeSelection interface is implemented by the UI library and is not intended to be implemented by consumer code. 

public interface ITreeNodeSelection : IReadOnlyCollection<ITreeNode>, IEnumerable<ITreeNode>, IEnumerable

Examples

The following example demonstrates how to iterate through the currently selected nodes in the tree and how to check whether a specific node is part of the current selection.

foreach (ITreeNode node in tree.NodeSelection)
{
    Console.WriteLine(node.Text);
}

if (tree.NodeSelection.Contains(someNode))
{
    Console.WriteLine("The node is selected.");
}

Remarks

Provides a read-only collection of selected ITreeNode instances.

The selection acts as the primary source for:

  • Drag-and-drop operations
  • Cut, Copy, and Paste operations

The collection reflects the current selection state of the tree and cannot be modified directly. Selection is managed by the ITree implementation.
