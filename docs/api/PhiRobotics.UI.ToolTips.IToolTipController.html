Describes a tooltip controller which provides functionality to manage and display tooltips for controls.

This interface is implemented by the PhiRobotics UI library and is not intended to be implemented by consumer code. 

public interface IToolTipController : IDisposable, IComponent, IDisposable

Remarks

The IToolTipController allows you to customize the appearance and behavior of tooltips displayed for controls and their elements.

You can configure visual aspects as well as behavior (such as display delays, visibility duration, and positioning).

The controller supports both regular tooltips and rich tooltips (IRichToolTip).

Controls that support tooltips expose a tooltip controller, which can be used to show, hide, and customize tooltips for those controls.

Consumer code can obtain an instance of IToolTipController using the CreateController() method. The lifetime of the tooltip controller instance is managed by the consumer.
