Describes an interface for a panel that displays alert windows.

The IAlertPanel interface is implemented by the PhiRobotics UI library and is not intended to be implemented by consumer code.

Examples

The following example shows how to create and display an alert panel:

IAlertPanel alertPanel = alertPanelFactory.CreatePanel();
alertPanel.PanelTextClick += AlertPanel_PanelTextClick;
alertPanel.Show(this, "Main Form", "Learn more...");

private void AlertPanel_PanelTextClick(object sender, AlertPanelTextClickEventArgs e)
{
    //...
}

Remarks

The IAlertPanel interface represents a UI component responsible for displaying alert windows to the user.

Instances of IAlertPanel are created using the CreatePanel() method.

The lifetime of an IAlertPanel instance is managed by the consumer. The consumer is responsible for disposing the panel when it is no longer needed.

Property Value

bool

Remarks

If enabled and the hot-tracked text is clicked, the PanelTextClick event is raised.

If disabled, clicking the alert text will not trigger the PanelTextClick event.

AllowHtmlText

Property Value

bool

Remarks

If the AllowHtmlText property is enabled, you can use HTML tags to format an alert window’s caption and text.

AutoFormDelay

Gets or sets the delay, in milliseconds, before an alert window automatically closes.

int AutoFormDelay { get; set; }

Property Value

int

Remarks

If the user does not hover over the alert window for the specified time, the window is automatically closed.

The user can prevent automatic closing by clicking the Pin button. The availability of this button is controlled by the ShowPinButton property.

Methods

Show(Form, IAlertPanelInfo, bool)

Displays an alert window displaying information from the specified IAlertPanelInfo.

void Show(Form owner, IAlertPanelInfo info, bool autoWidth = false)

Parameters

owner Form

An owner of the window. In a multi-monitor system, if the owner is set to null, the window will be displayed on the main monitor. Otherwise, the window will be displayed on the same monitor as the owner.

info IAlertPanelInfo

An IAlertPanelInfo object that contains settings used to initialize the newly created alert window.

autoWidth bool

Determines whether the width of the alert window is automatically adjusted to fit its content. The text will be displayed on a single line.

Show(Form, string, string, bool)

Displays an alert window with the specified caption and text.

void Show(Form owner, string caption, string text, bool autoWidth = false)

Parameters

owner Form

An owner of the window. In a multi-monitor system, if the owner is set to null, the window will be displayed on the main monitor. Otherwise, the window will be displayed on the same monitor as the owner.

caption string

A string that specifies the window’s caption.

text string

A string that specifies the window’s text, displayed when the text region is not hovered over.

autoWidth bool

Determines whether the width of the alert window is automatically adjusted to fit its content. The text will be displayed on a single line.

Remarks

If the AllowHtmlText property is enabled, you can use HTML tags to format an alert panel’s caption and text.

You can handle the BeforeFormShown event to customize the newly created alert window before its appearance on-screen.
