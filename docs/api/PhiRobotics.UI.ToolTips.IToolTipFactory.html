Namespace

Provides methods to create tooltip-related objects and access the default tooltip controller used across the application.

This interface is implemented by the PhiRobotics UI library and is not intended to be implemented by consumer code. 

public interface IToolTipFactory

Remarks

An instance of IToolTipFactory can be accessed via ToolTipFactory property.

Properties

DefaultController

Gets the default tooltip controller used to display tooltips across the application.

IToolTipController DefaultController { get; }

Property Value

IToolTipController

An instance of IToolTipController that manages the appearance and behavior of tooltips by default.

Remarks

The default controller ensures a consistent look and behavior for tooltips across all controls.

If specific controls require different tooltip behavior or appearance, create a separate controller using the CreateController() method and customize it as needed.

DefaultController's proerties are readonly and setter of the properties throws an exception InvalidOperationException.
