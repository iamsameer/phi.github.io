Defines a factory responsible for creating input editor instances.

The IInputEditorFactory interface is implemented by the UI library and is not intended to be implemented by consumer code. 

public interface IInputEditorFactory

Remarks

This factory provides a centralized way to create UI editors for different data types. Each editor type is identified by an InputEditorId.

Consumers can either use the generic Create(InputEditorId) method or dedicated methods such as CreateTextEdit() and CreateDateTimeEdit() for strongly-typed editor creation.

The lifetime of created editors is managed by the consumer. Each call returns a new, independent instance.

Properties

SupportedTypes

Gets the collection of input editor types that are supported by this factory.

IReadOnlyCollection<InputEditorId> SupportedTypes { get; }

Property Value

IReadOnlyCollection<InputEditorId>

Remarks

This collection represents the complete set of editor types that can be successfully passed to Create(InputEditorId). Consumers can use this information to determine which input editor is appropriate for a given scenario.

Returns

A newly created IInputEditor instance.

Remarks

Each call to this method returns a new, independent input editor instance. The lifetime of the returned editor is owned by the caller.

Implementations must guarantee that the returned object:

  • Inherits Control.
  • Implements IInputEditor.
  • Is fully initialized and ready for use.
