Interface IInputBoxArgs

Represents configuration settings for an IInputBox.

Namespace: PhiRobotics.UI.Dialogs
Assembly: PhiRobotics.UI.dll
Syntax
public interface IInputBoxArgs
Remarks

Instances of IInputBoxArgs are created using CreateInputBoxArguments(IInputEditor).

This interface encapsulates all settings required to configure an IInputBox, including caption, prompt, buttons, default values, and the associated editor.

The configured arguments are passed to CreateInputBox(IInputBoxArgs) to create an IInputBox instance.

The Editor property is initialized during creation and cannot be modified.

Examples

The following example demonstrates how to create and configure input box arguments.

// Obtain factory service
IUIFactoryService uiFactoryService = new UIFactoryService();

// Get required factories
ICommonDialogFactory commonDialogFactory = uiFactoryService.CommonDialogFactory;
IInputEditorFactory inputEditorFactory = uiFactoryService.InputEditorFactory;

// Create and configure editor
ITextInputEditor textEditor = inputEditorFactory.CreateTextEdit();
textEditor.Configure(typeof(string));

// Create input box arguments
IInputBoxArgs inputBoxArgs = commonDialogFactory.CreateInputBoxArguments(textEditor);
inputBoxArgs.Caption = "Input";
inputBoxArgs.Prompt = "Enter a value";
inputBoxArgs.DefaultResponse = "Default text";
inputBoxArgs.Buttons = new[] { System.Windows.Forms.DialogResult.OK, 
System.Windows.Forms.DialogResult.Cancel };
inputBoxArgs.DefaultButtonIndex = 0;
