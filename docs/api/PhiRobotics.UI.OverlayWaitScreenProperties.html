Image

Gets or sets a custom rotating image.

public Image Image { get; set; }

Property Value

Image

Remarks

If the IndicatorType property is set to Image, the screen displays a rotating image as a wait indicator. The default image depends on the skin. Use the ImageSize property to specify the size of the default image. The default image size depends on the size of the overlapped control. The Image property specifies a custom image.
