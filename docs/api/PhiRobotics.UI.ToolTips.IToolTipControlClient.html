Describes an interface for controls that provide custom tooltip information for their visual elements and child controls.

public interface IToolTipControlClient

Examples

The following example demonstrates how to implement the IToolTipControlClient interface and associate it with multiple controls to provide context-sensitive tooltips.

In this example, a single tooltip client is shared between a Button and a CheckBox. The tooltip text is dynamically updated based on the control currently under the cursor.

private void ToolTipControlClientExample(IToolTipFactory toolTipFactory)
{
    ToolTipControlClient controlClient = new ToolTipControlClient(toolTipFactory);

    // Register controls with the tooltip controller
    controller.AddClientControl(button1, controlClient);
    controller.AddClientControl(checkBox1, controlClient);
}

class ToolTipControlClient : IToolTipControlClient
{
    private readonly IToolTipControlInfo toolTipInfo;

    public ToolTipControlClient(IToolTipFactory factory)
    {
        // Create and initialize tooltip info
        toolTipInfo = factory.CreateControlInfo();
        toolTipInfo.Title = "Control";
        toolTipInfo.ToolTipType = ToolTipTypes.Standard;
    }

    public bool ToolTipsEnabled
    {
        get { return true; }
    }

    public IToolTipControlInfo GetToolTipInfo(Control clientControl, Point point)
    {
        // Update tooltip content based on the control type
        if (clientControl is Button button)
        {
            toolTipInfo.Text = "Button: " + button.Text;
            toolTipInfo.Object = clientControl;
            return toolTipInfo;
        }
        else if (clientControl is System.Windows.Forms.CheckBox checkBox)
        {
            toolTipInfo.Text = "CheckBox: " + checkBox.Text;
            toolTipInfo.Object = clientControl;
            return toolTipInfo;
        }

        // No tooltip for other controls
        return null;
    }
}

Remarks

After a control is registered via AddClientControl(Control, IToolTipControlClient), the IToolTipController monitors mouse activity over that control and invokes GetToolTipInfo(Control, Point) as needed.

The GetToolTipInfo(Control, Point) method is invoked only when ToolTipsEnabled is true.

note

This interface is intended to be implemented by custom or third-party controls that wish to integrate with the PhiRobotics tooltip infrastructure.

Remarks

If this property is false, GetToolTipInfo(Control, Point) is not invoked by the IToolTipController.

Methods

Returns tooltip information for the specified control at the given cursor position.

IToolTipControlInfo GetToolTipInfo(Control clientControl, Point point)

Parameters

clientControl Control

The control for which the tooltip is requested.

point Point

A Point specifying the cursor position in client coordinates relative to clientControl.

Returns

IToolTipControlInfo

An instance of IToolTipControlInfo containing tooltip data, or null if no tooltip should be displayed.

Remarks

This method is invoked by the IToolTipController when the instance of this interface is added to it via AddClientControl(Control, IToolTipControlClient) and when the mouse moves/hovers over the Control that has been added along with this instance.

Use CreateControlInfo() to create the returned object. Ensure that the returned instance has its Object property set and at least one of Title or Text assigned; otherwise, the tooltip will not be displayed.
