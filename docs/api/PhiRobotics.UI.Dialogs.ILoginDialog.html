Interface ILoginDialog

Namespace

Represents a user interface contract for a login dialog, providing support for credential input and dialog behavior.

The ILoginDialog interface is implemented by the UI library and is not intended to be implemented by consumer code. 

public interface ILoginDialog

Examples

The following example demonstrates how to display a login dialog and handle user actions.

ILoginDialog loginDialog = new LoginDialog();

loginDialog.EnableSignUpButton = true;

loginDialog.ForgotPasswordClick += (s, e) =>
{
    // Open password recovery page
};

loginDialog.SignupButtonClick += (s, e) =>
{
    // Open signup page
};

if (loginDialog.ShowDialog(this) == System.Windows.Forms.DialogResult.OK)
{
    bool remember = loginDialog.Remember;
    // Proceed with authentication
}

Remarks

Implementations of this interface are responsible for displaying the login UI and interacting with the user.

The dialog provides configurable options such as remembering user credentials and enabling sign-up functionality. It also exposes events to handle user actions like password recovery and account creation.

Instances of ILoginDialog are created using CreateLoginDialog(ILoginHandler).

The lifetime of an ILoginDialog instance is managed by the consumer. The consumer is responsible for disposing the dialog when it is no longer needed.

Properties

EnableSignUpButton

Gets or sets a value indicating whether the "Signup" button is enabled.

bool EnableSignUpButton { get; set; }

Property Value

bool

Remarks

This property can be used to conditionally enable or disable the "Signup" button in the user interface. If set to true, the "Signup" button will allow interaction with the "Signup" button.

Remember

Gets or sets a value indicating whether the system should remember the user's credentials.

bool Remember { get; set; }

Property Value

bool

Remarks

If set to true, the application may save the user's credentials for future sessions. This typically enables automatic login for the user in subsequent app launches.
