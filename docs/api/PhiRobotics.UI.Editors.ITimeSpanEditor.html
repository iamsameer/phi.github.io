Examples

The following example demonstrates how to configure a date-time editor:

ITimeSpanEditor editor = factory.CreateTimeSpanEdit();

editor.NullValuePrompt = "Enter date...";
editor.NullValuePromptForeColor = Color.LightSkyBlue;

ITimeSpanEditorMaskProperties maskProperties = editor.MaskProperties;
maskProperties.UseAdvancingCaret = true;
maskProperties.SpinWithCarry = true;
maskProperties.MaskExpression = "mm MM, ss SS";

Remarks

The ITimeSpanEditor interface extends IInputEditor and provides functionality for editing time interval values.

This editor supports structured input using time span masks, allowing users to enter and modify individual components such as days, hours, minutes, seconds, and fractional seconds.

Use the MaskProperties property to configure the input mask and control how time span values are entered and displayed.

Instances of ITimeSpanEditor are created using the IInputEditorFactory.
