Describes a base interface for input editors.

The IInputEditor interface is implemented by the UI library and is not intended to be implemented by consumer code.

Examples

The following example demonstrates how to create a text editor:

IInputEditor editor = inputEditorFactory.Create(InputEditorId.TextEdit);
ITextInputEditor textEditor = (ITextInputEditor)editor;

Remarks

The IInputEditor interface defines common functionality for all editor controls that allow users to view and modify values.

Editors expose their value through the EditValue property and provide events to track value changes before and after they occur.

In addition to value handling, this interface provides support for validation feedback, error indication, and tooltip integration.

Instances of IInputEditor are created using the Create(InputEditorId) method or dedicated factory methods for creating specific editor types.

The lifetime of an IInputEditor instance is managed by the consumer. Consumers are responsible for disposing the editor when it is no longer needed.

Remarks

The event data is provided by InputChangingEventArgs, which exposes the following properties:

  • OldValueGets the current editor value.
  • NewValue Gets or sets the value that is about to be assigned to the editor. Setting this property is not supported when the editor uses masked input.
note

Avoid displaying modal dialogs or changing focus within this event, as it may cause the editor to lose its current value.
