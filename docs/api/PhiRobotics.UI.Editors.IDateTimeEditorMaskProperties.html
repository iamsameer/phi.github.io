Defines properties that control date-time masking behavior for text-based editors.

The IDateTimeEditorMaskProperties interface is implemented by the UI library and is not intended to be implemented by consumer code.

Examples

The following examples demonstrate configuring a date-time editor mask:

// Using a text editor with a date-time mask
ITextInputEditor editor = inputEditorFactory.CreateTextEdit();

editor.Configure(null, EditMaskType.DateTime);
IDateTimeEditorMaskProperties maskProperties = 
    (IDateTimeEditorMaskProperties)editor.MaskProperties;

maskProperties.UseAdvancingCaret = true;
maskProperties.SpinWithCarry = true;
maskProperties.MaskExpression = "f";

// Using a dedicated date-time editor
IDateTimeEditor dateTimeEditor = inputEditorFactory.CreateDateTimeEdit();
IDateTimeEditorMaskProperties maskProperties = dateTimeEditor.MaskProperties;

maskProperties.UseAdvancingCaret = true;
maskProperties.SpinWithCarry = true;
maskProperties.MaskExpression = "f";

Remarks

This interface extends ITextEditorMaskProperties to provide additional functionality specific to date and time input.

properties control advanced editing behaviors such as automatic caret movement between segments and value propagation across adjacent segments.

The lifetime of an IDateTimeEditorMaskProperties instance is managed by the associated editor. Consumers typically do not create or dispose this object directly.

Properties

SpinWithCarry

Gets or sets whether a neighboring segment value is automatically adjusted when the current segment value is incremented or decremented. (users can spin values with editor buttons or keyboard arrow keys)

bool SpinWithCarry { get; set; }

Property Value

bool

true to increase or decrease the neighboring segment value when the user spins through values of a segment; otherwise, false.

Remarks

For example, consider a mask HH:mm (hours and minutes) with a value of 00:59.

  • If this property is false, incrementing minutes results in 00:00.
  • If this property is true, incrementing minutes results in 01:00.

Similarly, decrementing from 01:00 results in either 01:59 or 00:59, depending on this setting.

UseAdvancingCaret

Gets or sets whether the caret automatically moves to the next segment when the current segment is fully populated.

bool UseAdvancingCaret { get; set; }

Property Value

bool

true to advance the caret to the next segment when input is complete; otherwise, false.

Remarks

This behavior improves data entry efficiency by reducing the need for manual navigation between segments.
