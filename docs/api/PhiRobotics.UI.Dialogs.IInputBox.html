Represents a modal dialog that allows the user to enter a value using a single editor.

The IInputBox interface is implemented by the UI library and is not intended to be implemented by consumer code.

Examples

The following example demonstrates how to create and display an input box that accepts a string value from the user.

// Obtain factory service
IUIFactoryService uiFactoryService = new UIFactoryService();

// Get required factories
ICommonDialogFactory commonDialogFactory = uiFactoryService.CommonDialogFactory;
IInputEditorFactory inputEditorFactory = uiFactoryService.InputEditorFactory;

// Create and configure editor
ITextInputEditor textEditor = inputEditorFactory.CreateTextEdit();
textEditor.Configure(typeof(string));

// Create input box arguments
IInputBoxArgs inputBoxArgs = commonDialogFactory.CreateInputBoxArguments(textEditor);
inputBoxArgs.Caption = "Input";
inputBoxArgs.Prompt = "Enter a value";

// Create and show input box
using (IInputBox inputBox = commonDialogFactory.CreateInputBox(inputBoxArgs))
{
    if (inputBox.Show(this) == System.Windows.Forms.DialogResult.OK)
    {
        object value = inputBox.InputResult;
        // Process value
    }
}

Remarks

Instances of IInputBox are created using CreateInputBox(IInputBoxArgs).

The lifetime of an IInputBox instance is managed by the consumer. The consumer is responsible for disposing the dialog when it is no longer needed.
