Examples

The following example demonstrates how to configure a text editor:

ITextInputEditor editor = factory.CreateTextEdit();

editor.NullValuePrompt = "Enter value...";
editor.NullValuePromptForeColor = Color.LightSkyBlue;
editor.BeepOnError = true;
editor.UseMaskAsDisplayFormat = false;

Remarks

The ITextInputEditor interface extends IInputEditor and provides functionality for editing textual values with optional masking and formatting.

This editor supports multiple input modes through mask configuration, allowing you to restrict input, enforce formats, and control how values are displayed and stored.

Use the Configure(Type, EditMaskType) method to define the value type and mask behavior. Depending on the configuration, the editor exposes different MaskProperties implementations.

Instances of ITextInputEditor are created using the IInputEditorFactory.

Methods

Configure(Type, EditMaskType)

Configures the editor to accept values of the specified valueType and applies the specified mask.

void Configure(Type valueType = null, EditMaskType mask = EditMaskType.None)

Parameters

valueType Type

A Type that specifies the type of values accepted by the editor. If null, the editor accepts string values.

mask EditMaskType

An EditMaskType value that specifies the mask type to apply.

Remarks

When valueType is null, the editor is configured for string input and the MaskProperties instance is initialized based on the specified mask.

When valueType is not null, the editor initializes mask settings appropriate for the specified type (for example, numeric or date/time input).
