Represents a tooltip item used as a title or footer in a rich tooltip.

This interface is implemented by the PhiRobotics UI library and is not intended to be implemented by consumer code.

Remarks

This item is typically used to display emphasized text, such as a header or summary section.

Inherits all features of IRichToolTipItem, including support for text, images, and formatting, but is rendered using a distinct visual style appropriate for titles or footers.
