Defines properties that control numeric masking behavior for text-based editors.

The INumericEditorMaskProperties interface is implemented by the UI library and is not intended to be implemented by consumer code.

Examples

The following examples demonstrate configuring a numeric editor mask:

// Using a text editor with a numeric mask
ITextInputEditor editor = inputEditorFactory.CreateTextEdit();

editor.Configure(typeof(Byte), EditMaskType.Numeric);
INumericEditorMaskProperties maskProperties = 
    (INumericEditorMaskProperties)editor.MaskProperties;

maskProperties.AutoHideDecimalSeparator = true;
maskProperties.HideInsignificantZeros = true;

// Using a dedicated numeric editor
INumericEditor numericEditor = inputEditorFactory.CreateNumericEdit<Byte>();

numericEditor.MaskProperties.MaskExpression = "d";
numericEditor.MaskProperties.AutoHideDecimalSeparator = true;
numericEditor.MaskProperties.HideInsignificantZeros = true;

Remarks

This interface extends ITextEditorMaskProperties to provide functionality specific to numeric input.

It enables users to enter and edit numeric values such as integers and floating-point numbers.

These properties control how numeric values are displayed and interpreted, including handling of decimal separators, insignificant zeros, and value behavior during editing operations.

The INumericEditor uses this interface to expose numeric-specific mask configuration through its mask properties.

The lifetime of an INumericEditorMaskProperties instance is managed by the associated editor. Consumers typically do not create or dispose this object directly.

Properties

AutoHideDecimalSeparator

Gets or sets whether the decimal separator is hidden when the value is a whole number.

bool AutoHideDecimalSeparator { get; set; }

Property Value

bool

true to hide the decimal separator for whole numbers; otherwise, false.

Remarks

This property improves readability by removing unnecessary decimal separators when no fractional part is present.

HideInsignificantZeros

Gets or sets whether insignificant trailing zeros are hidden.

bool HideInsignificantZeros { get; set; }

Property Value

bool

true to hide insignificant zeros; otherwise, false.

Remarks

For example, a value such as 10.500 may be displayed as 10.5 when this property is enabled.

IsValueAfterDeleteZeroThenNull

Gets or sets whether the editor value becomes zero before becoming null when all digits are deleted.

bool IsValueAfterDeleteZeroThenNull { get; set; }

Property Value

bool

true to set the value to 0 after the last digit is removed, and then to null on the next delete; otherwise, false to set the value directly to null.

Remarks

When a user presses Backspace or Delete, this property determines the resulting value after the last digit is removed.

If enabled, the value transitions from a numeric value → 0 → null. If disabled, the value transitions directly from a numeric value → null.

If the entire content is cleared at once (for example, using Ctrl+Delete or selecting all text and deleting), the value becomes null immediately, regardless of this setting.
