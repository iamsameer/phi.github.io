Interface IMessageBox

Represents a dialog used to display informational, warning, or confirmation messages.

The IMessageBox interface is implemented by the UI library and is not intended to be implemented by consumer code.

Remarks

Instances of IMessageBox are created using CreateMessageBox().

The message box can be displayed using overloads of the Show method,either by passing simple parameters or a fully configured IMessageBoxArgs instance.

The lifetime of an IMessageBox instance is managed by the consumer. The consumer is responsible for disposing the dialog when it is no longer needed.

Examples

The following example demonstrates how to create and display a message box using arguments.

// Obtain factory service
IUIFactoryService uiFactoryService = new UIFactoryService();

// Get common dialog factory
ICommonDialogFactory commonDialogFactory = uiFactoryService.CommonDialogFactory;

// Create message box arguments
IMessageBoxArgs args = commonDialogFactory.CreateMessageBoxArguments();
args.Caption = "Information";
args.Content = "Operation completed successfully.";
args.Buttons = new[] { System.Windows.Forms.DialogResult.OK };
args.MessageBoxIcon = System.Windows.Forms.MessageBoxIcon.Information;

// Create and show message box
using (IMessageBox messageBox = commonDialogFactory.CreateMessageBox())
{
    messageBox.Show(args);
}
