Represents configuration settings for an IMessageBox.

public interface IMessageBoxArgs

Examples

The following example demonstrates how to create and configure message box arguments.

// Obtain factory service
IUIFactoryService uiFactoryService = new UIFactoryService();

// Get common dialog factory
ICommonDialogFactory commonDialogFactory = uiFactoryService.CommonDialogFactory;

// Create arguments
IMessageBoxArgs args = commonDialogFactory.CreateMessageBoxArguments();
args.Caption = "Confirmation";
args.Content = "Are you sure you want to continue?";
args.Buttons = new[] 
{ 
    System.Windows.Forms.DialogResult.Yes, 
    System.Windows.Forms.DialogResult.No 
};
args.DefaultButtonIndex = 0;
args.MessageBoxIcon = System.Windows.Forms.MessageBoxIcon.Question;

Remarks

Instances of IMessageBoxArgs are created using CreateMessageBoxArguments().

This interface encapsulates all settings required to configure a message box, including caption, content, buttons, icons, sounds, and optional behaviors.

The configured arguments can be passed to Show(IMessageBoxArgs) to display the dialog.
