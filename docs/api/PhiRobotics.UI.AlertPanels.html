Interfaces

IAlertForm

Represent an alert window.

The IAlertForm interface is implemented by the PhiRobotics UI library and is not intended to be implemented by consumer code.

IAlertPanel

Describes an interface for a panel that displays alert windows.

The IAlertPanel interface is implemented by the PhiRobotics UI library and is not intended to be implemented by consumer code.

IAlertPanelFactory

Defines a factory for creating alert panels and related components.

The IAlertPanelFactory interface is implemented by the PhiRobotics ui framework and is not intended for consumer implementation.
