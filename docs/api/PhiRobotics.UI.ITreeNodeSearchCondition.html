Represents a condition used to determine whether a tree node matches specific search criteria.

public interface ITreeNodeSearchCondition

Examples

The following example demonstrates how to implement a condition that finds nodes whose text contains a specific value.

public class NodeTextContainsCondition : ITreeNodeSearchCondition
{
    private readonly string text;

    public NodeTextContainsCondition(string text)
    {
        this.text = text;
    }

    public bool IsMatch(ITreeNode node)
    {
        return node.Text.Contains(text, StringComparison.OrdinalIgnoreCase);
    }
}

// Usage
ITreeNodeSearchCondition condition = new NodeTextContainsCondition("Program");
IReadOnlyCollection<ITreeNode> nodes = tree.Find(condition, true);

Remarks

Implement this interface to define custom search logic when locating nodes using methods such as Find(ITreeNodeSearchCondition, bool).

The condition is evaluated for each ITreeNode encountered during the search operation. Nodes for which IsMatch(ITreeNode) returns true are included in the result.
