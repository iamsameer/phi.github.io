Represents a collection of items that define the content regions of a rich tooltip.

This interface is implemented by the PhiRobotics UI library and is not intended to be implemented by consumer code. 

public interface IToolTipItemCollection : IEnumerable

IEnumerable.GetEnumerator()

Remarks

The Items property provides access to this collection.

Each item in the collection represents a visual region (such as title, content, or separator) within the tooltip.

Items are displayed vertically in the tooltip, in the same order in which they are added to the collection.
