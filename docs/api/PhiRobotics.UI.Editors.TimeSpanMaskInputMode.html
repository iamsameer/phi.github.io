Lists the input modes of time span edit mask.

public enum TimeSpanMaskInputMode

Fields

NotRestricted = 1

Users can enter any number into any mask part. If the resulting value is out of the range limit, its neighboring segment is automatically increased.

NotRestrictedLargestUnit = 2

Users can enter any values to the largest mask part only. For remaining mask segments, this mode behaves in a manner similar to “Restricted” mode.

Restricted = 0

Users are allowed to enter only valid numbers for each mask segment.

Remarks

The InputMode property uses the TimeSpanMaskInputMode enumeration values.
