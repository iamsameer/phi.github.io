Interface ICommonDialogFactory

Provides factory methods for creating common dialog components such as message boxes, input boxes, and login dialogs.

The ICommonDialogFactory interface is implemented by the UI library and is not intended to be implemented by consumer code.

Namespace: PhiRobotics.UI.Dialogs
Assembly: PhiRobotics.UI.dll
Syntax
public interface ICommonDialogFactory
Remarks

Use CommonDialogFactory to obtain an instance of this interface. The lifetime of the factory instance is managed by the UI library.

This factory centralizes the creation of commonly used dialogs and their associated argument objects.

Instances of IInputBox, IMessageBox, ILoginDialog created by this factory are owned by the consumer. The consumer is responsible for managing their lifetime and disposing them when they are no longer needed.

CreateLoginDialog(ILoginHandler)

Creates a new instance of ILoginDialog using the specified login handler.

Declaration
ILoginDialog CreateLoginDialog(ILoginHandler handler)
Parameters
Type Name Description
ILoginHandler handler

An implementation of PhiRobotics.ILoginHandler that provides the authentication logic for the dialog.
Returns
Type Description
ILoginDialog

A new instance of ILoginDialog.
Remarks

The provided handler is used by the dialog to retrieve user credentials and perform authentication when the user attempts to log in.

This design separates UI concerns from authentication logic, allowing custom login implementations to be plugged into the dialog.

Examples
ILoginHandler handler = new LoginHandler();
ILoginDialog dialog = factory.CreateLoginDialog(handler);

if (dialog.ShowDialog(parentForm) == DialogResult.OK)
{
    // Login successful
}
