Represents a collection of buttons displayed within alert windows.

public interface IButtonCollection

Examples

The following example demonstrates how to create and add custom buttons to an IAlertPanel and handle their interaction.

// Create alert panel.
IAlertPanel alertPanel = alertPanelFactory.CreatePanel();

// Create a regular custom button.
IButton button1 = alertPanelFactory.CreateButton("buttonOpen");
button1.Hint = "Open file";
button1.Image = Properties.Resources.Folder_16x16;

// Create a check custom button.
IButton button2 = alertPanelFactory.CreateButton("buttonAlert");
button2.IsCheckStyle = true;
button2.Down = true;
button2.Hint = "Alert On";
button2.Image = Properties.Resources.Timer_16x16;

// Add buttons to the AlertControl
alertPanel.Buttons.Add(button1);
alertPanel.Buttons.Add(button2);

// Subscribe to the events to process button clicks
alertPanel.ButtonClicked += AlertPanel_ButtonClicked;
alertPanel.ButtonDownChanged += AlertPanel_ButtonDownChanged;

// Show a sample alert window.
IAlertPanelInfo panelInfo = alertPanelFactory.CreatePanelInfo("New Window", "Text");
alertPanel.Show(this, panelInfo);

Remarks

Alert windows can display custom buttons, typically shown at the bottom of the panel. To include custom buttons, create IButton instances using CreateButton(string) and add them to the Buttons collection.

The buttons contained in this collection are applied only to newly created alert windows. Modifying the collection does not affect buttons already displayed in existing alert windows.
