PhiRobotics Alert Panel UI

The PhiRobotics Alert Panel provides a flexible and extensible way to display non-intrusive notification windows (alerts) in your application. These alerts can include text, images, and interactive elements such as buttons and hyperlinks.

Overview

An alert panel is a UI component responsible for displaying temporary notification windows to the user. It supports:

  • Customizable content (caption, text, images)
  • Interactive elements (buttons, hyperlinks)
  • Animation and positioning
  • Event-driven customization
  • Multiple concurrent alerts with queuing

Architecture

The PhiRobotics Alert Panel UI is composed of the following core components:

Component Description
IAlertPanel Main interface used to display and manage alert windows
IAlertPanelFactory Factory for creating alert panels, buttons, and panel info
IAlertPanelInfo Defines the content and metadata of an alert
IAlertForm Represents an individual alert window instance
IButton Represents a button within an alert window
IButtonCollection Collection of buttons associated with an alert panel

Creating and Displaying Alerts

IAlertPanel alertPanel = alertPanelFactory.CreatePanel();
alertPanel.AutoHeight = true;
alertPanel.Show(this, "Update Available", "Click to install.");

Using AlertPanelInfo

IAlertPanelInfo info = alertPanelFactory.CreatePanelInfo("Download Complete", "Click to open file");
info.Image = Properties.Resources.SuccessIcon;
info.Tag = filePath;

IAlertPanel alertPanel = alertPanelFactory.CreatePanel();
alertPanel.AutoHeight = true;
alertPanel.Show(this, info);

Custom Buttons

IButton openButton = alertPanelFactory.CreateButton("open");
openButton.Hint = "Open file";
openButton.Image = Properties.Resources.OpenFileIcon;

IButton toggleButton = alertPanelFactory.CreateButton("toggle");
toggleButton.IsCheckStyle = true;
toggleButton.Down = true;
toggleButton.Image = Properties.Resources.ToggleIcon;

IAlertPanel alertPanel = alertPanelFactory.CreatePanel();
alertPanel.AutoHeight = true;
alertPanel.Buttons.Add(openButton);
alertPanel.Buttons.Add(toggleButton);
alertPanel.Show(this, "Update Available", "Click to install.");

Handling Button Events

alertPanel.ButtonClicked += (s, e) =>
{
    if (e.ButtonName == "open")
    {
        // Handle open action
    }
};

alertPanel.ButtonDownChanged += (s, e) =>
{
    if (e.ButtonName == "toggle")
    {
        bool isEnabled = e.Down;
    }
};

The IAlertPanel interface exposes several events for customization:

  • BeforeFormShown
  • PanelFormLoad
  • PanelTextClick
  • FormClosing
  • ButtonClicked
  • ButtonDownChanged
  • HyperLinkClicked
  • QueryFormWidth

Summary

The PhiRobotics Alert Panel UI provides:

  • Lightweight notification system
  • Rich customization options
  • Interactive UI elements
  • Event-driven extensibility
