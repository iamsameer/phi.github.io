PhiRobotics Fly-Out Panel UI

Overview

The PhiRobotics Fly-Out Panel provides a flexible way to display contextual UI elements that can be dynamically attached to a host control. A fly-out panel behaves like a lightweight container that can be shown, hidden, repositioned, and resized at runtime.

This infrastructure is suitable for:

  • Contextual tool panels
  • Pop-up configuration views
  • Assistant or helper panels

Architecture

The PhiRobotics Fly-Out Panel UI is composed of the following core components:

Component Description
IFlyOutPanel Represents the fly-out panel instance.
IFlyOutPanelFactory Responsible for creating fly-out panel instances.
FlyOutPanelOptions Defines configuration settings used during panel creation.
AnchorStyles Specifies how the panel is positioned relative to its host control.
ResizeMode Specifies how the panel responds to resize operations.

Creating a Fly-Out Panel

Fly-out panels are created using the IFlyOutPanelFactory, which can be obtained from:

IUIFactoryService.FlyOutPanelFactory

Example

// Create a fly-out panel without initial content
IFlyOutPanel panel = flyOutPanelFactory.CreatePanel(null);

// Create content
Control content = new CustomControl();
content.Size = new Size(250, 100);

// Assign content
panel.Content = content;

// Attach to host control
panel.AttachTo(hostControl);

// Show panel
panel.ShowPanel();

Using FlyOutPanelOptions

You can customize panel behavior using FlyOutPanelOptions.

Example

FlyOutPanelOptions options = new FlyOutPanelOptions(
    anchorStyle: AnchorStyles.BottomRight,
    resizeMode: ResizeMode.Both,
    disposeContent: true,
    disposeContentOnChange: true,
    enableIndent: true);

IFlyOutPanel panel = flyOutPanelFactory.CreatePanel(content, options);
panel.AttachTo(hostControl);
panel.ShowPanel();

Positioning (AnchorStyles)

The AnchorStyles enum determines how the panel is positioned relative to the host control.

  • TopLeft
  • TopRight
  • BottomRight
  • BottomLeft

Resizing (ResizeMode)

The ResizeMode enum determines how the panel responds to resize operations.

  • None
  • Height
  • Width
  • Both

Resize is performed using:

panel.ResizePanel(deltaWidth, deltaHeight);

Attaching and Displaying

A panel must be attached before it can be displayed.

panel.AttachTo(hostControl);
panel.ShowPanel();

Showing at a Specific Location

Point? actualLocation = panel.ShowPanel(new Point(100, 200));

Moving the Panel

If enabled, the panel can be repositioned:

panel.IndentPanel(deltaX, deltaY);

Visibility Control

panel.ShowPanel();
panel.HidePanel();

Disposal Behavior

  • DisposeContent
  • DisposeContentOnChange

Lifecycle

The lifetime of the fly-out panel is managed by the consumer.

panel.Dispose();
