Working with Input Editors

Input editors provide a consistent way to view and edit values.
They support different data types such as text, numbers, boolean values, date-time, time span, and colors.

Creating Editors

Use the IInputEditorFactory to create editor instances.
You can either use a generic creation method or strongly-typed helper methods.

Using Strongly-Typed Methods

ITextInputEditor textEditor = factory.CreateTextEdit();
IDateTimeEditor dateTimeEditor = factory.CreateDateTimeEdit();
ITimeSpanEditor timeSpanEditor = factory.CreateTimeSpanEdit();
IBooleanEditor booleanEditor = factory.CreateBooleanEdit();
IColourEditor colorEditor = factory.CreateColourEdit();

Using Generic Creation

IInputEditor editor = factory.Create(InputEditorId.TextEdit);

You can cast the returned editor to a specific type when needed:

ITextInputEditor textEditor = (ITextInputEditor)editor;

Common Features

All input editors share common functionality defined by IInputEditor.

Value Handling

Editors expose their value through the EditValue property:

editor.EditValue = "Sample Text";
var value = editor.EditValue;

Validation and Events

Editors provide events to track value changes:

  • ValueChanging → Raised before the value changes\
  • ValueChanged → Raised after the value changes

Placeholder Text

editor.NullValuePrompt = "Enter value...";
editor.NullValuePromptForeColor = Color.LightGray;

Working with Masks

Text Editor Mask

ITextInputEditor editor = factory.CreateTextEdit();
editor.Configure(null, EditMaskType.RegEx);
editor.MaskProperties.MaskExpression = "\\d+";

Date-Time Editor Mask

IDateTimeEditor editor = factory.CreateDateTimeEdit();
var mask = editor.MaskProperties;
mask.UseAdvancingCaret = true;
mask.SpinWithCarry = true;
mask.MaskExpression = "f";

Time Span Editor Mask

ITimeSpanEditor editor = factory.CreateTimeSpanEdit();
var mask = editor.MaskProperties;
mask.UseAdvancingCaret = true;
mask.SpinWithCarry = true;
mask.MaskExpression = "c";

Numeric Editors

INumericEditor intEditor = factory.CreateNumericEdit<int>();
intEditor.EditValue = 10;

Boolean Editors

IBooleanEditor editor = factory.CreateBooleanEdit();
editor.EditValue = true;

Color Editors

IColourEditor editor = factory.CreateColourEdit();
editor.EditValue = Color.Red;

Lifetime Management

Each call to the factory creates a new editor instance.

  • Editors are independent
  • The consumer is responsible for disposing them

Choosing the Right Editor

Scenario Editor Type
Free text input ITextInputEditor
Numeric values INumericEditor
True/false values IBooleanEditor
Date and time IDateTimeEditor
Time intervals ITimeSpanEditor
Color selection IColourEditor

Summary

  • Use IInputEditorFactory to create editors
  • All editors implement IInputEditor
  • Use strongly-typed editors for better usability
  • Configure masks for structured input
  • Manage editor lifetime in your application
